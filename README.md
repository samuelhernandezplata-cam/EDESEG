Este repositorio contiene el código fuente del sitio web de EDESEG, desarrollado para ofrecer una plataforma moderna, segura y responsiva que permita presentar información institucional, servicios, noticias y canales de contacto para clientes y visitantes.

El proyecto está diseñado para proporcionar una experiencia de usuario intuitiva, optimizada para dispositivos móviles y de escritorio, siguiendo buenas prácticas de desarrollo web.

Objetivos
Presentar la información institucional de EDESEG.
Mostrar el catálogo de servicios ofrecidos.
Facilitar el contacto con clientes y usuarios.
Brindar una interfaz rápida, accesible y responsiva.
Servir como base para futuras ampliaciones del sitio.
Características
Diseño responsivo.
Navegación intuitiva.
Optimización para SEO.
Compatible con los principales navegadores.
Código organizado y modular.
Fácil mantenimiento y escalabilidad.
